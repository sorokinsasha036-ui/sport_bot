# Telegram Sport Bot

Це **Telegram-бот**, створений мовою **Python**, який допомагає працювати зі звітами про різні види спорту.

---

## Функції бота
- Додавання нового виду спорту
- Пошук спорту
- Видалення спорту
- Перегляд списку видів спорту

---

## Технології

Python — основна мова програмування

Aiogram — бібліотека для створення Telegram-ботів

Telegram Bot API — для роботи бота з Telegram

GitHub — для збереження та відстеження коду

FSM (Finite State Machine) — для керування діалогами та станами бота

---

## Команди

```plaintext
/start       — запуск бота
/help        — список команд
/add_sport   — додати новий вид спорту
/search_sport — знайти вид спорту
/delete_sport — видалити спорт
/sport       — показати список видів спорту
